## Research standard

For every scheme or tool, capture:

- name
- jurisdiction
- owner/funder
- official URL
- status/date checked
- supported intervention types
- eligibility rules
- payment rates if applicable
- evidence requirements
- maintenance obligations
- useful API/data sources if any
